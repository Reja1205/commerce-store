## Environment Variables

### Backend (.env)

MONGODB_URI=your_mongodb_connection
JWT_SECRET=your_secret
PORT=4000

### Frontend (.env.local)
NEXT_PUBLIC_API_URL=https://your-backend-url.onrender.com

## Run Locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
